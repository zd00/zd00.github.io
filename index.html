Hexo+Github pages免费建站

deng 2019-12-18

捣鼓了两天，终于把自己的blog 搭起来了，和我一样的新手小白可以参考一下。

网站框架： Hexo+Github Pages
本地编辑内容、hexo生成静态文件推送到github仓库托管来实现互联网访问

Github Pages设置

GitHub Pages 作用是介绍托管在GitHub的项目，由于空间免费稳定，所以常被用来做搭建个人网站。

  1. 注册github账号，新建仓库并命名为 username.github.io

注意：（前缀username 是你注册的账号ID）
点击文字链接可进入相应页面

  1. 安装Git（用于本地提交代码到github仓库）
    Windows下载 默认安装就行，打开GitBash可进入命令行终端。可参考详细教程

此时可写个md文件推送到github进行网页简单测试：（可略过）

  • 新建文件夹blog并初始化作为Git代码库
    $ git init blog
  • 在工作区blog创建编辑展示的md文件并提交到暂存区
  • 将暂存区提交到版本库并连接到远程仓库进行推送部署
  • 关于git更多信息 感兴趣的可以看看 git概念与常用命令、git参考资料*

ok 现在你可以通过访问userid.github.io访问创建的展示页面了！
ヾ(๑╹◡╹)ﾉ”

PS: Github pages 可绑定独立域名进行访问
