VScode+七牛云图床

使用VScode作为代码编辑工具，开源、页面友好。最主要支持的扩展插件多啊，应该是目前最流行的Windows平台应用程序的集成开发环境。不多说，开搞！

  • VScode下载安装
  • 添加汉化扩展插件(‘ctrl+shift+x’ >输入chinese >插件安装启用)

    建议添加的插件: Markdown Preview Enhanced（更直观预览你编写的内容，即上图右边区域）

图床选择有很多，腾讯云COS、七牛云存储、阿里云OSS、SM.MS、微博等等；图床工具用的PicGo（VScode扩展里有Picgo插件支持，但老不记得的快捷键没用习惯；后来又安装了windows.exe版本,别嘲笑我(；′⌒`)）

  • PicGo图床工具需要的参数

¶七牛云图床

  • 注册账号
  • 新建存储空间（获得存储空间名、访问网址）
    个人面板>密钥管理 （获得AK、SK）
    华南存储区域是z2
